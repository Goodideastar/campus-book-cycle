项目数据结构设计

1. 用户表 (campus_book_users)
id: String 用户ID，使用时间戳
username: String 用户名
password: String 加密后的密码
userType: String 用户类型：student 或 admin
phone: String 手机号
email: String 电子邮箱
address: String 收货地址（旧字段，已迁移到addresses）
addresses: Array 收货地址数组
id: String 地址ID
name: String 收货人姓名
phone: String 联系电话
province: String 省份
city: String 城市
district: String 区县
detail: String 详细地址
isDefault: Boolean 是否默认地址
avatar: String 头像URL
createdAt: String 创建时间
updatedAt: String 更新时间
2. 书籍表 (campus_book_books)
id: String 书籍ID，使用时间戳
name: String 书籍名称
author: String 作者
cateName: String 分类名称
originalPrice: Number 原价
salePrice: Number 售价
status: String 状态：onSale, sold, offSale
auditStatus: String 审核状态：unaudit, audited, rejected
publisher: String 发布人用户名
cover: String 封面图片URL
desc: String 书籍描述
createdAt: String 创建时间
updatedAt: String 更新时间
3. 交易表 (campus_book_transactions)
id: String 交易ID，使用时间戳
bookId: String 书籍ID
bookName: String 书籍名称
buyer: String 买家用户名
seller: String 卖家用户名
price: Number 价格
quantity: Number 数量
status: String 状态：pending, completed, canceled
tradeType: String 交易类型：offline, online
paymentMethod: String 支付方式：wechat, alipay
address: String 交易地址（线下交易）
phone: String 联系电话（线下交易）
shippingAddress: Object 收货地址信息
id: String 地址ID
name: String 收货人姓名
phone: String 联系电话
province: String 省份
city: String 城市
district: String 区县
detail: String 详细地址
createTime: String 创建时间
4. 通知表 (campus_book_notifications_{username})
id: String 通知ID，使用时间戳
title: String 通知标题
description: String 通知描述
time: String 通知时间
read: Boolean 是否已读
5. 购物车表 (campus_book_cart_{username})
bookId: String 书籍ID
quantity: Number 数量
6. 轮播图表 (campus_book_carousel)
id: String 轮播图ID，使用书籍ID
7. 登录状态 (campus_book_login)
值: Boolean 登录状态，true表示已登录
8. 当前用户信息 (campus_book_user)
存储当前登录用户的完整信息，结构同用户表
